# DSU AI Academic Advisor

An Agentic AI project developed during the FDP on Agentic AI & LLM Engineering.

## Features

- Local LLM using Ollama (Qwen3)
- Local Embedding Model (Nomic Embed)
- ChromaDB Vector Database
- Retrieval-Augmented Generation (RAG)
- AI/ML Curriculum Knowledge Base

## Technology Stack

- Python
- Ollama
- Qwen3
- Nomic Embed Text
- ChromaDB
- Jupyter Notebook
